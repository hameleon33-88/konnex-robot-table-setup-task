# Setup the Table – Robotics AI Task

This repository contains a robotics AI task model
designed for the Konnex simulation environment.

## Task
Setup the table.

## Objective
The robot must organize and arrange dining objects
on a table in preparation for a meal.

## Environment
A simulated dining area with tableware items.

## Expected Robot Actions
- Detect table surface
- Identify plates, forks, knives and glasses
- Place tableware in correct positions
- Align objects according to standard dining layout

## Evaluation
The agent is evaluated based on:

- object recognition accuracy
- placement precision
- task completion efficiency

## Platform
Konnex AI Miner Testnet
